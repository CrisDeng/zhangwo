# 掌握 - OpenClaw x QQ， 图形化界面一键安装，无需 CLI 操作。

基于 OpenClaw 的本地私人助手，无编码经验的朋友可 2 分钟接入到 QQBot，通过 QQBot 给助理下发指令。

## 📖 使用教程

详细的安装和使用教程请查看：**[使用教程](https://docs.qq.com/doc/DSkFCZ3hVWVBuT2Jy)**

## 🖥️ 系统要求

- macOS 15.0 (Sequoia) 或更高版本
- 支持 Apple Silicon (M1/M2/M3) 和 Intel 芯片

## 📥 下载安装

从 [Releases](https://github.com/CrisDeng/zhangwo/releases) 页面下载最新版本的 DMG 文件，打开后将"掌握"拖入 Applications 文件夹即可。

## 🚀 快速开始

1. 下载并安装应用
2. 打开"掌握"应用
3. 按照引导完成配置
4. 开始与 AI 助手对话
